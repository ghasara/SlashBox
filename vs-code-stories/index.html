Stories - New VS-Code Extension

This is not just like instagram, youtube, whatsapp, facebook or any social media stories that you have seen.



Do you ever just wake up in the morning, open up VS Code, and think to yourself " Wow I wonder what other coders are doing around the world, what are they coding on ? ".
- This is how Ben Awad introduce stories to everyone.


This is very interesting to see what anyone is capable of making, an stories extension for Visual Studio Code.

Ben Awad created an extension for VS Code that you can use to view Instagram like stories and upload your own ones for others to see.

If you haven't already check out the extension then here is the link to Ben Awad youtube video where he announces the launch of VS-Code stories Click here.

All the codes to the extensions is open source, do check it out Click here.
